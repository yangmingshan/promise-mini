# promise-mini
一个极度小巧的 Promise polyfill，支持完整的 ES2015 Promise API

## 体积
0.5KB (min + gzip)

## Node.js
安装：

```
npm install promise-mini
```

使用：

```
var Promise = require('promise-mini');
```
